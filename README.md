🎤 Real-Time Speech Recognition (Day 20)

This project is part of my Day 20 of Mini Project Series, where I build one project daily to improve my development skills.

🚀 Project Overview

This is a Real-Time Speech Recognition Web App built using HTML, CSS, and JavaScript.
It converts spoken words into text instantly using the browser's Speech Recognition API.

---

✨ Features

- 🎙️ Converts voice to text in real-time
- 🔄 Continuous listening (auto-restart)
- 📄 New line for each final sentence
- 🖊️ Editable text area
- 📜 Auto-scroll as text grows
- 🐞 Console logging for debugging

---

🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Web Speech API (SpeechRecognition)

---

📂 How It Works

1. The app accesses the microphone using the browser API.
2. Speech is captured and processed in real time.
3. The transcript is displayed instantly on the screen.
4. Each final sentence appears on a new line.
5. The app automatically restarts listening when it stops.

---

💡 Key Concepts Used

- DOM Manipulation
- Event Listeners
- Speech Recognition API
- Real-time data handling
- Auto-scroll implementation
